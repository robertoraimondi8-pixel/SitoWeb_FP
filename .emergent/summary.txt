<analysis>**original_problem_statement**:
The user has requested a major new feature: a Tornei (Tournaments) module. This module must exist separately from the current Leghe (Leagues) system but should reuse the existing prediction and scoring engine.

PRODUCT REQUIREMENTS for Tornei:
-   **Creation**: Tournaments are created only by a Super Admin via the admin panel.
-   **Configuration**: Admin can set the number of participants (8, 16, 32), duration in matchdays, and structure (e.g., 4 groups of 4, top 2 advance).
-   **Structure**: The system must support a group stage followed by a single-elimination knockout bracket.
-   **Enrollment**: Users can see and join available tournaments from a new Tornei section in the app. Enrollment is initially free but should be designed to support future payment integration.
-   **Automation**: The system should automatically generate groups when the tournament starts and create the knockout bracket based on group stage results and admin-defined rules (e.g., 1st vs 2nd).
-   **Gameplay**:
    -   Matchups are 1v1. The winner of a matchup is the user with the higher total prediction score for that matchday.
    -   Group stage points: Win = 3, Draw = 1, Loss = 0.
    -   The admin selects a different set of matches for each tournament matchday.
    -   Prediction submission window is the same as for leagues (closes 1 minute before the first match).
-   **Live View**: A critical feature is a live matchup screen showing the two competing users, their predictions, and the score updating in real time.
-   **UI**: A new Tornei section should be added to the app, separate from Leghe. It should have sub-sections for I miei tornei (My Tournaments) and Iscriviti a nuovi tornei (Join New Tournaments). The home screen should also feature a banner for available tournaments.

**User's preferred language**: Italiano
